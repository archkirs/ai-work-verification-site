About

A reviewer should not need the whole archive first.

AI-assisted work can produce many process materials: AI chats, drafts, screenshots, source notes, versions, references, and final files. Sending everything at once can be hard to review.

A Verification Code Registry tests a simpler idea: a compact record and code that let a reviewer see the main process summary first, then understand what evidence exists and what the record cannot prove.

FAQ

A code can help navigation. It does not replace review.

In this pilot, the registry idea is a lightweight test: can a code and compact record make process evidence easier to share and review?

The pilot does not independently determine legal ownership, copyright status, publication rights, academic acceptance, platform compliance, or originality.
